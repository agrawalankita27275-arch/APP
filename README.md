# Task Organizer and Meal Tracker


A productivity and wellness application that helps users manage daily tasks and track meals in one place. The app allows users to prioritize tasks, monitor task completion progress, and maintain meal records for better organization and healthy habits.

## Features

### Task Planner

* Create, edit, and delete tasks.
* Assign priority levels (High, Medium, Low) to tasks.
* Track task completion status.
* Monitor overall task progress.
* Organize daily activities efficiently.

### Meal Tracker

* Add and manage daily meals.
* Record meal details and schedules.
* View meal history.
* Track eating habits over time.

### Progress Tracking

* Save completed tasks automatically.
* Display task completion progress.
* Help users stay productive and achieve goals.


## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/task-planner-meal-tracker.git
```

2. Navigate to the project directory:

```bash
cd task-planner-meal-tracker
```

3. Install dependencies:

```bash
npm install
```

or

```bash
flutter pub get
```

4. Run the application:

```bash
npm start
```

or

```bash
flutter run
```

## Usage

1. Create tasks and assign priorities.
2. Mark tasks as completed when finished.
3. Track your completion progress.
4. Add meals throughout the day.
5. Review meal history and task performance.
6. Task reminders and notifications.
7. Meal nutrition tracking.
